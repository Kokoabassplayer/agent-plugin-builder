Workflow

From idea to installable repo.

  1. Name it.Create a plugin package with the right folder layout.
  2. Write the skill.Put the reusable agent behavior in SKILL.md first.
  3. Wire adapters.Expose the same capability to each harness without rewriting the core.
  4. Ship proof.Run checks, publish docs, and give users one command to install.
